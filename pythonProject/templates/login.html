Welcome to the InfoCheck Web Page

InfoCheck aims to alert companies that the things they may be putting online are not safe.
InfoCheck is a web scraping tool built in Python that scrapes information from websites
in order to examine how much information any such company is putting online.
InfoCheck is designed to simulate how an attacker will behave in regards to the MITRE ATT&CK Framework.

About InfoCheck

InfoCheck was built with Python and Gazpacho for web scraping,
when using InfoCheck you should be able to see
page titles, headings, phone numbers and emails that appear
on the website of the URL you enter.
Results will appear showing you what was found on each page of the website.
